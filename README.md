# Hans Rubber Industries — Premium Industrial Enterprise Website

A premium cinematic industrial manufacturing website built using Next.js, TypeScript, Tailwind CSS, and Framer Motion.

The project focuses on:
- cinematic enterprise UI
- industrial luxury aesthetics
- editorial layouts
- responsive architecture
- smooth motion design
- fullstack contact form integration

---

# Tech Stack

## Frontend
- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Backend
- Next.js API Routes
- REST API architecture
- JSON request handling
